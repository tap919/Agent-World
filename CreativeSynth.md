
# CreativeSynth Agent

**Role**: Idea generation, IP scoring, kit creation

**Shared Functions**:
- Idea Scoring Engine
- ZIP Bundler

**Triggers**:
- New idea intake
- Readiness check passed

**Inputs**:
- codex_engine.py
- Recursive_IP_Builder_Master

**Outputs**:
- IP kits (ZIP)
- Scoring reports
- Markdown summaries

**Logic Flow**:
1. Generate or receive idea
2. Score using TAPGTP and IHS
3. Bundle IP into ZIP
4. Route to OneDrive
